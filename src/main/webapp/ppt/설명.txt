안녕하세요 저희 프로젝트 이름은 이젠호텔입니다.

간단하게 목차를 소개드리면 프로젝트 개요, 프로젝트 구조(ERD), 화면 별 주요기능, 프로젝트 시연

저희 팀은 가상의 '이젠 호텔'이라는 호텔 예약 시스템의 기능을 가지는 사이트를 구현해보았습니다.

저희 팀은 저를 포함한 5명으로 역할분담을 하였는데 서로가 배운 과정이 다르지 않기 때문에 
이번 프로젝트에서는 수업한 내용을 복기한다는 생각으로 각자가 맡은 기능을 처음부터 끝까지
작성하기로 하였습니다. 

보여지는 페이지 구성, DAO와 SERVLET등 자신이 맡은 기능에 필요한 부분을 스스로 작성하기로 했고

각자가 맡은 기능을 말씀드리면

(이민혁)
저는 메인 페이지에서 보이는 간단한 홈페이지 안내와 hearder.jsp, footer.jsp 페이지를 작성하여 
모든 페이지 마다 노출되도록 하였고 상단에 header 부분에서는 각 기능으로 이동하도록 연결하였습니다.
주요 맡은 기능으로는 '예약' 부분을 구현하였고 팀원이 만든 기능을 하나의 프로젝트에 합쳐서 테스트
하는 역할을 하였습니다.

백준석님은 로그인과 회원가입 페이지와 실제로 데이터베이스에 접근해 로그인 검증과 회원가입 등록을
구현하였습니다.

김진님은 이용후기 기능을 담당하셨는데 기존의 게시판 기능을 활용해서 댓글과 대댓글도 가능하도록
구현하였습니다. 그 외로 PPT 문서 작업을 병행하였습니다.

심영섭님은 이젠호텔의 객실정보를 안내하는 페이지를 작성하고 프로젝트에 사용할 호텔 관련 자료를
수집하였습니다.

황인철님은 데이터베이스의 구조(ERD)를 작성하고 관리자에게 문의하는 기능과 예약번호를 통해 
예약 내용을 확인하는 기능을 구현하였습니다.


개발환경은 윈도우10 운영체제를 사용하였고 개발툴은 이클립스 데이터베이스는 오라클
개발언어로는 자바와 자바스크립트를 위주로 코드를 작성하였고 자바는 11버전을 사용하였습니다.
그리고 서블릿 컨테이너로는 톰캣을 사용하였습니다.

스케줄은 기획시간은 굉장히 짧았고 프로젝트를 만들 시간이 촉박했기 때문에 개발과 테스트를
빠른 주기로 반복하면서 기능을 완성하였습니다. 

ERD는 호텔, 객실, 회원, 예약, 이용후기, 문의 등 기능의 개수만큼 작성하였고 각각 외래키를 
생성하여 관계를 설정하였습니다. 

화면 구성에서는 전체 페이지의 사이트맵을 작성하였습니다.

페이지의 모양은 기능 개발이 끝난 이후에 지속적으로 업데이트를 했기 때문에 조금
다를 수 있습니다.

회원가입에서는 입력 칸이 모두 채워져야 하도록 설정했으며 각각의 입력마다 검증을 구현하였고 
회원가입의 아이디 중복체크 부분은 input type이 text인 입력창에 addEventListener의 keyup 이벤트
가 발생하면 ajax로 서블릿을 호출하여 입력한 문자열이 데이터베이스 아이디에 존재하는지를 체크하고
아이디 입력창 아래에 적절한 안내 문구를 띄워주었습니다.

(내가 작성한 내용 우측에 내용 표시)

로그인 화면에서도 마찬가지로 모든 입력창을 작성하게끔 하고 아이디가 존재하지 않을 경우와
아이디가 존재하지만 비밀번호가 틀릴 경우를 구분해서 경고창을 띄워주었습니다. 

객실정보 페이지에서는 화면에 마우스 버튼을 누르면 사진 모양이 변한다던가 하는 
동적으로 보여지는 부분과 상세 설명 내용을 작성하였습니다. 그리고 자바스크립트를
사용하여 각 이미지가 자동으로 넘어가거나 버튼에 의해 넘어가도록 구현했습니다.

객싱정보 상세 내용에서 예약 버튼을 클릭할 경우 실제 예약 페이지로 넘어가고 
만약 로그인이 되지 않은 상태라면 로그인폼 페이지로 넘어가게 됩니다.

(예약 우측 보면서 설명)



예약 내역 확인에서는 데이터베이스에 들어있는 예약번호를 기준으로 테이블을 조회하여 
ajax를 사용하여 바로 페이지에 표시되도록 기능을 구현하였습니다.


문의하기 내용에서는 작성내용을 입력하면 관리자만 확인할 수 있다는 가정하에 
데이터베이스에 저장까지만 기능을 구현하였습니다.



이용후기 게시판에서는 수업 때 배운 게시판기능, 댓글, 대댓글의 기능을 작성했고
게시물의 삭제, 수정도 가능하도록 하였습니다.
접근 권한을 구분하는 기능은 시간 관계상 구현하지 못했고,
로그인 후 이용이 가능하도록 구현하였습니다


실제로 사이트가 동작하는 방식을 간단한 설명과 함께 시연하겠습니다.
